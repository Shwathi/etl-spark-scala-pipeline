# etl-spark-scala-pipeline
# ETL Pipeline using Scala and Apache Spark

## Overview
This project demonstrates building a scalable ETL pipeline using Apache Spark with Scala.

## Features
- Data extraction from CSV files
- Data cleaning and transformation
- Aggregation using Spark SQL functions
- Output stored in Parquet format

## Tech Stack
- Scala
- Apache Spark
- SBT

## How to Run
1. Install Spark and SBT
2. Run Main.scala
